# eEssence Experience Knowledge Base

## Purpose

This knowledge base contains the intellectual property behind every eEssence experiential learning programme.

It is not an activity library.

It is a collection of facilitator knowledge, participant behaviour, learning patterns and programme design principles accumulated through years of practical facilitation.

Mission AI reads this knowledge to recommend, sequence and facilitate programmes.

Every new activity should be represented by a single YAML file.

Each file should describe:

- Activity information
- Learning objectives
- Participant behaviour
- Facilitator observations
- Intervention triggers
- Debrief questions
- CodeShift links
- Experience tags

The knowledge base is the intelligence.

Mission AI is the interface.
